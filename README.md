# Calculadora de Materiais para Obra Residencial

Projeto em Java para Desenvolvimento de Sistemas.

## Funcionalidades
- Cálculo do volume de concreto para vigas
- Cálculo da quantidade de tijolos
- API REST organizada em camadas
- Interface rm hrml

## Tecnologias
- Java 17
- Spring Boot
- Spring Web
- Maven


A aplicação vai em:
- API: `http://localhost:8080`
- swagger - http://localhost:8080/swagger-ui/index.html
